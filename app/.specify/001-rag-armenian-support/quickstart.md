# Quickstart: RAG Agent with Armenian Language Support

This document provides a quick overview of how to use the RAG agent with Armenian language support.

## 1. Running the Application

1.  Ensure you have all the required dependencies installed from `requirements.txt`.
2.  Run the application using `uvicorn app.main:app --reload`.

## 2. Interacting with the Agent

1.  Open the web interface in your browser.
2.  Click the "Start Mic" button and speak to the agent in Armenian.
3.  The agent will respond in Armenian based on the information in its custom knowledge base.

## 3. Updating the Knowledge Base

1.  Use the "Upload PDF" button on the web interface to select a PDF document.
2.  The document will be uploaded, and the knowledge base will be updated automatically.
3.  You can then ask the agent questions about the content of the uploaded PDF.
